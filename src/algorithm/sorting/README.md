> 问题：快速排序与归并排序异同点？
1. 都是分治思想，都使用递归；
2. 快排在拆的过程中进行排序，归并先拆再合并，合并过程中排序；

笼统的讲：
快排以基准值拆分，大的放右边，相等的放中间，小的放左边，这就是排序的过程，拆完后即排好序了。
归并以集合中间索引拆成左右两部分，两部分是无序的，再对左右两部分递归拆分，一直拆到左右两部分各剩一个元素，然后各部分排序，再合并，合并完成后即排好序了。
